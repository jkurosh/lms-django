# تغییرات جدید در قالب بیماری‌های داخلی

## تغییرات اعمال شده

### 1. تغییر چیدمان اصلی
- **قبل**: تست‌ها سمت چپ، گزینه‌ها سمت راست
- **بعد**: آزمایش‌ها سمت چپ، گزینه‌ها سمت راست

### 2. ساختار جدید آزمایش‌ها
آزمایش‌ها حالا به شکل دکمه‌های جداگانه نمایش داده می‌شوند:

#### دکمه‌های آزمایش:
- **CBC** (آزمایش خون کامل)
- **Clinical Chemistry** (بیوشیمی بالینی)
- **تصاویر تشخیصی** (رادیوگرافی، سونوگرافی، اکو)
- **سایر تست‌ها** (تست‌های دیگر)

### 3. نحوه کارکرد جدید
1. **انتخاب دکمه**: کاربر روی دکمه آزمایش مورد نظر کلیک می‌کند
2. **نمایش نتایج**: نتایج آن گروه آزمایش نمایش داده می‌شود
3. **تغییر گروه**: با کلیک روی دکمه دیگر، گروه جدید نمایش داده می‌شود

### 4. ویژگی‌های جدید
- **دکمه‌های تعاملی**: با انیمیشن و افکت‌های بصری
- **نمایش تعداد تست‌ها**: هر دکمه تعداد تست‌های موجود را نشان می‌دهد
- **دکمه فعال**: دکمه انتخاب شده با رنگ سبز مشخص می‌شود
- **نمایش تدریجی**: نتایج با انیمیشن fadeIn نمایش داده می‌شوند

### 5. گروه‌بندی خودکار
سیستم به طور خودکار تست‌ها را بر اساس نامشان گروه‌بندی می‌کند:

```javascript
// گروه‌بندی بر اساس کلیدواژه‌ها
- CBC: شامل "cbc", "خون", "complete blood"
- Chemistry: شامل "chemistry", "بیوشیمی", "biochemistry"
- Imaging: شامل "رادیوگرافی", "سونوگرافی", "اکو", "x-ray", "ultrasound", "ecg"
- Other: سایر تست‌ها
```

### 6. استایل‌های جدید
- **دکمه‌های آزمایش**: طراحی مدرن با گرادیانت
- **نتایج آزمایش**: کارت‌های جداگانه برای هر تست
- **ریسپانسیو**: سازگار با موبایل و تبلت

### 7. فایل‌های تغییر یافته
- `internal-diseases.html`: تغییر ساختار HTML
- `internal-diseases.js`: اضافه کردن منطق جدید
- `styles.css`: اضافه کردن استایل‌های جدید

## نحوه استفاده

### برای کاربران:
1. انتخاب مطالعه موردی
2. کلیک روی دکمه آزمایش مورد نظر (CBC، Chemistry، تصاویر)
3. مطالعه نتایج نمایش داده شده
4. انتخاب گزینه پاسخ در سمت راست

### برای توسعه‌دهندگان:
برای اضافه کردن تست جدید، کافی است در فایل JavaScript تست را اضافه کنید. سیستم به طور خودکار آن را گروه‌بندی می‌کند.

## مزایای جدید
- **سازماندهی بهتر**: تست‌ها به صورت منطقی گروه‌بندی شده‌اند
- **تجربه کاربری بهتر**: دسترسی آسان‌تر به تست‌های مختلف
- **نمایش واضح‌تر**: هر گروه تست به صورت جداگانه نمایش داده می‌شود
- **انعطاف‌پذیری**: امکان اضافه کردن گروه‌های جدید
