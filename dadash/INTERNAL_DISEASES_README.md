# صفحه بیماری‌های داخلی دامپزشکی

## توضیحات کلی

این صفحه بخشی از اپلیکیشن مطالعه مورد دامپزشکی است که به طور خاص بر روی بیماری‌های داخلی و سیستمیک در حیوانات تمرکز دارد.

## ویژگی‌های اصلی

### 1. زیردسته‌بندی‌های بیماری‌های داخلی
- **بیماری‌های گوارشی**: مشکلات دستگاه گوارش، کبد و لوزالمعده
- **بیماری‌های قلبی و عروقی**: مشکلات قلب، عروق و فشار خون
- **بیماری‌های تنفسی**: مشکلات ریه و مجاری تنفسی
- **بیماری‌های غدد درون‌ریز**: مشکلات هورمونی و متابولیک
- **بیماری‌های کلیوی**: مشکلات کلیه و سیستم ادراری
- **بیماری‌های کبدی**: مشکلات کبد و مجاری صفراوی

### 2. مطالعات موردی تعاملی
هر زیردسته‌بندی شامل مطالعات موردی واقعی با:
- **تاریخچه بیمار**: اطلاعات کامل بیمار و علائم
- **تست‌های تشخیصی**: آزمایش‌های خون، تصویربرداری و غیره
- **گزینه‌های پاسخ**: انتخاب تشخیص صحیح
- **توضیحات آموزشی**: نکات کلیدی و منابع

### 3. قابلیت‌های تعاملی
- **تصاویر قابل بزرگنمایی**: کلیک روی تصاویر برای مشاهده بهتر
- **سیستم امتیازدهی**: پیگیری پیشرفت کاربر
- **توضیحات تفصیلی**: برای هر پاسخ و تشخیص

## ساختار فایل‌ها

### `internal-diseases.html`
- ساختار HTML صفحه
- منوی ناوبری
- بخش‌های مختلف مطالعات موردی

### `internal-diseases.js`
- منطق تعاملی صفحه
- داده‌های نمونه مطالعات موردی
- مدیریت رویدادها و تعامل کاربر

### `styles.css`
- استایل‌های مشترک
- طراحی ریسپانسیو
- انیمیشن‌ها و افکت‌ها

## نحوه استفاده

### برای دانشجویان دامپزشکی:
1. **انتخاب زیردسته‌بندی**: بر اساس موضوع مورد علاقه
2. **مطالعه تاریخچه**: خواندن کامل اطلاعات بیمار
3. **بررسی تست‌ها**: تحلیل نتایج آزمایش‌ها و تصاویر
4. **انتخاب تشخیص**: انتخاب بهترین گزینه
5. **مطالعه توضیحات**: یادگیری نکات کلیدی

### برای اساتید:
1. **اضافه کردن مطالعات موردی جدید**: ویرایش فایل JavaScript
2. **تنظیم سطح دشواری**: beginner, intermediate, advanced
3. **اضافه کردن تصاویر**: لینک به تصاویر جدید
4. **به‌روزرسانی منابع**: اضافه کردن منابع جدید

## نمونه مطالعات موردی موجود

### بیماری‌های گوارشی:
1. **پانکراتیت حاد در سگ**
   - سطح: متوسط
   - زمان: 25 دقیقه
   - تمرکز: تشخیص و درمان پانکراتیت

2. **IBD در گربه**
   - سطح: پیشرفته
   - زمان: 30 دقیقه
   - تمرکز: تشخیص بیماری التهابی روده

### بیماری‌های قلبی و عروقی:
1. **نارسایی قلبی در سگ**
   - سطح: متوسط
   - زمان: 25 دقیقه
   - تمرکز: کاردیومیوپاتی اتساعی

## اضافه کردن مطالعات موردی جدید

برای اضافه کردن مطالعه موردی جدید:

```javascript
{
    id: 3,
    title: "عنوان مطالعه موردی",
    patientHistory: "تاریخچه بیمار...",
    difficulty: "متوسط", // beginner, intermediate, advanced
    estimatedTime: 20,
    tests: [
        {
            id: 1,
            name: "نام تست",
            description: "توضیح تست",
            result: "نتیجه تست",
            isRequired: true,
            sortOrder: 1
        }
    ],
    options: [
        {
            id: 1,
            text: "گزینه پاسخ",
            isCorrect: true,
            explanation: "توضیح گزینه"
        }
    ],
    explanation: {
        text: "توضیح کلی",
        keyLearningPoints: ["نکته 1", "نکته 2"],
        references: ["منبع 1", "منبع 2"]
    }
}
```
